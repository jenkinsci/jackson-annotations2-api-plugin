# Jackson Annotations 2 API Plugin

This plugin exposes the [Jackson Annotations 2](https://github.com/FasterXML/jackson) API to Jenkins plugins.

## Changelog

* See [GitHub Releases](https://github.com/jenkinsci/jackson-annotations2-api-plugin/releases) for recent versions

## Developer documentation

See the [Jenkins Developer Documentation](https://jenkins.io/doc/developer/book/).

Cheatsheet:

* To build the plugin locally: `mvn clean verify`
* To test in a local Jenkins instance: `mvn hpi:run`
